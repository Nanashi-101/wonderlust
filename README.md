# Wonderlust Expeditions

Wonderlust Expeditions is a premium, high-end travel and adventure web application built with **Next.js 16**. It is designed for explorers seeking breathtaking landscapes and meaningful journeys across Northern India.

![Wonderlust Preview](public/stock/hero-bg.jpg) *(Replace with actual screenshot if available)*

## ✨ Features

- **🌍 Multi-language Support**: Fully internationalized with support for 5 languages:
  - English
  - हिन्दी (Hindi)
  - বাংলা (Bengali)
  - Polski (Polish)
  - Français (French)
- **🎭 Smooth Animations**: Powered by **GSAP** and **Framer Motion** for a fluid, premium user experience.
- **📱 Responsive Design**: Fully optimized for mobile, tablet, and desktop experiences.
- **🏔️ Interactive Destinations**: Dynamic routing for exploring locations like Kashmir, Ladakh, Manali, and Rishikesh.
- **💼 Expedition Packages**: Detailed guides for high-altitude treks and spiritual retreats.
- **📸 Visual Storytelling**: A sophisticated gallery component with filtering capabilities.

## 🚀 Tech Stack

- **Framework**: [Next.js 16 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [GSAP](https://greensock.com/gsap/) & [Framer Motion](https://www.framer.com/motion/)
- **Internationalization**: [next-intl](https://next-intl-docs.vercel.app/)
- **Icons**: [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)
- **Components**: [Radix UI](https://www.radix-ui.com/) & [Shadcn UI](https://ui.shadcn.com/)
- **Swiper**: [Swiper.js](https://swiperjs.com/) for touch-friendly sliders.

## 🛠️ Getting Started

### Prerequisites

- Node.js 18.17 or later
- npm, yarn, pnpm, or bun

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Nanashi-101/wonderlust.git
   cd wonderlust
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open the application**:
   Navigate to [http://localhost:3000](http://localhost:3000) to see the result.

## 📁 Project Structure

- `app/[locale]/`: Main application pages with locale-based routing.
- `app/components/`: Reusable primitive and composite UI components.
- `messages/`: Translation dictionaries for all supported languages.
- `i18n/`: Internationalization configuration (routing, navigation, and request handlers).
- `public/`: Static assets including images and fonts.
